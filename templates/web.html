Cumulative deaths by COVID-19

{% if mobile %} {% else %} {{ script_plot1|safe }} {{ div_plot1|safe }} {% endif %}

Using days since the spread started (when deaths pass 0.0001% of the population) and number of deaths as a percentage of the population allows the state of virus spread between different countries. The target is a horizontal line - which means there are no new deaths. China is the best example of a country which shows a full recovery, The legitimacy of their data reporting is questionable however..

Acceleration of deaths

{% if mobile %} {% else %} {% endif %}

The increase of deaths reported per day over the last 7 days as a percentage of the countries population. A positive acceleration means the number of deaths being reported per day is increasing - the situation is getting worse. Countries that are recovering will have a negative acceleration. Countries with a near 0 acceleration have stabilised the situation and should start to recover. Some countries have different policies for what counts as a death by covid-19, for example until recently France only counted covid-19 deaths which occured in hospitals. This would distort that countries data compared to countries which included deaths from home too.

Acceleration of confirmed cases

{% if mobile %} {% else %} {% endif %}

The increase of confirmed cases reported per day over the last 7 days as a percentage of the countries population. Confirmed cases isn't a relaible way of comparing countries as every country has different test coverage, it could however give an indication of the virus spread slowing down.
